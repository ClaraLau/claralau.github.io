Project Overview

Dates: January-April 2019

Description:
The goal of this project was to apply interface design principles to create an improved interface for foreign language learning. Both team members had a passion for learning new languages but were frustrated in the difficulty staying motivated and finding learning platforms that were engaging. Therefore, this project focused on exploring how the mobile language learning experience could be improved.

Team Members: 1 fellow classmate in this undergraduate interface design course.

My Contributions:
I participated in all stages of the project including ideation, gathering design requiremenets, early design creation, design refinement and defining testing procedures.
